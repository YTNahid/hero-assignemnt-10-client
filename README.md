# AlphaReviews- Game Review Platform

Welcome to **AlphaReviews**, your ultimate destination for honest and insightful game reviews. Whether you're looking for the latest AAA blockbuster or a hidden indie gem, AlphaReviews helps you discover your next gaming adventure with ease.

## üìñ Description
AlphaReviews is a full-stack web application built with MERN, designed for gamers to share and explore game reviews. Users can browse highest-rated games, read detailed reviews, and contribute their own opinions to the community. The platform features a secure authentication system, allowing users to manage their personal reviews and keep track of games they want to play via a dedicated watchlist.

## ‚ú® Key Features
- **Dynamic Hero Section**: A stunning slider showcasing the core value of the platform using `Swiper`.
- **Top-Rated Games**: Instantly access the highest-rated reviews on the homepage.
- **Full CRUD Functionality**: Authenticated users can Add, View, Update, and Delete their own reviews.
- **Personal Watchlist**: Save interesting games to your private watchlist for future reference.
- **Secure Authentication**: Firebase-powered Login and Registration (including Google Sign-in).
- **Interactive UI**: 
  - Smooth animations with `React Awesome Reveal`.
  - Responsive layout built with `Tailwind CSS` and `DaisyUI`.
  - User-friendly notifications via `React Toastify`.
  - Confirmation modals using `SweetAlert2`.
  - Show text on hover using `React-tooltip`.
- **Filtering & Sorting**: (If implemented) Easily find games by genre or rating.
